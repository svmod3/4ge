02 work

Network Intrusion Detection

in progress

Signature-based IDS misses novel attacks. Can a supervised model classify malicious network flows without hand-written rules, and how well does it generalize to unseen zero-day exploits?

  • Binary classification on the NSL-KDD dataset (~148k network flows) evaluating the generalization gap.
  • Built an end-to-end inference pipeline using Random Forest, securing 97% precision on attacks.
  • Exposed the model as a production-ready, high-performance FastAPI web service for real-time traffic analysis.
  • Current best (Official Test Set): F1 0.76 / Recall 0.63 (Proves near-perfect 1.00 F1 on known patterns, highlighting the zero-day challenge).
  • Next: Neural Network (MLP) implementation, feature scaling refinement, and live packet sniffing via Scapy on Raspberry Pi.
